     toFixed(val,arg){

     }

## 小胡子语法 {{}}
## 指令
- v-model
  - checkbox  [value,....]
  - radio  val
  - select option
- v-text
- v-html
- v-once
- v-bind :
  - :class
    - [c1,c2,{box1:true}]
    - {box:c}
  - :style
    - {color:c1,fontSize:"20px"}
    - [s1,s2]
- v-on  @
- v-for (item,index) in 对象,数组,字符串,数字
- v-if   操作DOM
- v-else if
- v-else
- v-show display:"none"/"block"

##实例上的方法
- vm.$set()  数据响应
- vm.mount("#app"")  挂载 相当于 el: "#app"
- vm.$watch("监控的属性",{})
- vm.$data.属性   vm.属性
- vm.$el  #app
